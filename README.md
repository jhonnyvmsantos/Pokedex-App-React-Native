
# Pokedex

Repositório para armazenar o programa Pokédex (Pokedex).

Basicamente, o aplicativo se resume a uma coleção de todos (?) os pokémons do anime "Pokemon".

## Stack utilizada

**Front-End:** React, React-Native (TypeScript) e Expo.

**API:** PokéAPI.
